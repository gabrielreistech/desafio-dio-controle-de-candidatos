# desafio-dio-controle-de-candidatos

Este é um código básico que realiza a avaliação de candidatos para determinar se serão chamados para o trabalho. A avaliação é feita com base na pretensão salarial do candidato em comparação à base salarial da vaga.

## Funcionalidades

- Avaliação da pretensão salarial dos candidatos.
- Determinação se o candidato será chamado para a vaga.

## Como Executar

Para executar este projeto, basta clonar o repositório e rodar o código. O projeto é simples e pode ser iniciado com o comando `run`.
